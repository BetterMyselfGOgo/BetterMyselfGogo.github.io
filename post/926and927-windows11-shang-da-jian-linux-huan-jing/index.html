9.26&9.27 Windows11上搭建Linux环境

首先在windows中启用支持虚拟机功能（跟着网上的教程卡了很久，解决方式是再启用虚拟机平台）

然后在Windows store中下载Ubuntu，配置好自己的用户名与密码后即可使用。（需要下载一个新的内核）进而继续配置gcc,在ubuntu中输入sudo(获取管理员权限) apt install gcc,然后会出现error，解决方法是apt-get update,此时又会遇到update不了的情况，解决方法是利用su命令进入管理员模式，然后再进行install和update即可。

最后，再VSCode上下载ubuntu插件并且“connect with ubuntu”，就可以在VScode上使用ubuntu终端了。

遗留问题gcc: error: hello.c: No such file or directory时如何解决.
